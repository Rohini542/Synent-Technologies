# Login Page

A simple and responsive Login Page built using HTML, CSS, and JavaScript.

## Features

- Clean and user-friendly interface
- Email and password input fields
- Basic form validation
- Responsive design
- Modern UI layout

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure


Login Page/
│
├── index.html

## Purpose

This project was developed to practice front-end web development concepts such as form creation, styling, and JavaScript validation.

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any web browser.
