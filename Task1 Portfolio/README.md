# Personal Portfolio Website

A responsive personal portfolio website developed using HTML and CSS to showcase my profile, skills, projects, and contact information.

## Features

- Responsive design
- Professional profile section
- About Me section
- Skills showcase
- Project details
- Contact information
- Clean and modern user interface

## Technologies Used

- HTML5
- CSS3

## Project Structure

Portfolio/
│
├── index.html
├── style.css
└── README.md


## Purpose

This portfolio website was created to present my personal and academic information, technical skills, and projects in a professional manner.

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` or open it in any web browser.
